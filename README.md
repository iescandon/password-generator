# Live Site
https://iescandon.github.io/


## Description 

This website is designed as an application to help the user generate a random password after answering a few criteria questions. Criteria questions are asked via prompts and confirms which are triggered by clicking the "generate password" button. If prompts and confirms are not answered properly, they will get an alert letting them know what the error was and to input information again. New password is output directly onto the website. HTML and CSS is designed to be make the application easy to use and understand.

## Installation

In order to install this project, you must login to GitHub and go to github.com/iescandon/password-generator. Once there you will click on the green button that says clone or download. You will be given choices on how to download: using the ssh/html key or downloading the zip file.

Using SSH/HTML Key:
You will copy the link shown and open up either terminal (mac: pre-installed) or gitbash (pc: must be installed). Once the application is open, you will type git clone _paste url here_. Once you have cloned the git password-generator repo, cd into the repo and type "open ." to open the folder which contains all files used for the website. Once inside the folder, click on index.html to open the website in the browser.

Using Download ZIP:
Click on Download Zip. Locate the file and double click it to unzip the file. Locate the unzipped folder and and open it. All the files for the website will be within this folder. Click on index.html to open the website in the browser.


## Usage 

Website functions as a normal website. Once on the website simply click on the "generate password" button and fill in the responses or click okay to confirm.
